Leadory – Documentation

Thank you for choosing Leadory. This documentation will help you understand the file structure, customization options, and how to switch between color variants easily.

Overview

Introduction

Leadory is a modern, clean, and responsive business landing page template built with HTML, CSS, Bootstrap, and lightweight JavaScript. It is designed for startups, agencies, and professional websites.

Customization

Color Variants (Blue, Emerald, Orange)

Leadory uses a CSS variable-based color system. This allows you to change the entire theme color by editing only three variables inside style.css.

Open css/style.css and locate the :root section: 

/* Blue (Default) */
--primary-color: #4f46e5;
--primary-hover: #4338ca;
--primary-light: #eef2ff;

/* Emerald */
/* --primary-color: #10b981;
--primary-hover: #059669;
--primary-light: #ecfdf5; */

/* Orange */
/* --primary-color: #f97316;
--primary-hover: #ea580c;
--primary-light: #fff7ed; */

How to use a variant:

  • Only one color set should be active at a time
  • Comment out the unused variants
  • Uncomment the variant you want to use
  • Save the file and refresh the browser

This system ensures clean code, easy maintenance, and unlimited future color themes.

UX

Scroll Animations

Scroll animations are implemented using a lightweight JavaScript observer and CSS transitions.

To disable animations, simply remove the reveal class from the HTML elements.
